ベクトルとは？

大きさと向きの2つの情報を持つ量を学ぼう

①ベクトルとは何か

定義

ベクトルとは、① 大きさ　② 向き　の2つの情報を持っている量のこと。

  • 大きさ ……矢印（線）の 長さ
  • 向き ……矢印の 方向

例：「北へ 50km/h で移動する」という速度はベクトル。
一方「気温 25℃」のように向きのない量は スカラー と呼ぶ。

重要ポイント

大きさと向きが等しければ、位置が違っても同じベクトルとみなす。

（ベクトルは平行移動しても変わらない！）

すべての矢印は同じベクトル $\vec{a}$（大きさ・向きが等しい）

②ベクトルの表し方

点 A を始点、点 B を終点とする有向線分によるベクトルを

$$\overrightarrow{AB}$$

と表す。また、小文字に矢印をつけて $\vec{a}$ と表すこともある。

$\overrightarrow{AB}$ の大きさ（長さ）を 絶対値 といい、

$$|\overrightarrow{AB}|$$

と書く。

始点 A から終点 B への矢印が $\overrightarrow{AB}$

③逆ベクトルと零ベクトル

逆ベクトル

定義

$\overrightarrow{AB}$ と大きさが等しく、向きが逆のベクトルを 逆ベクトル といい、

$$-\overrightarrow{AB} = \overrightarrow{BA}$$

と表す。

零ベクトル

定義

始点と終点が一致するベクトル（大きさが 0 のベクトル）を 零ベクトル といい、$\vec{0}$ と表す。向きは定めない。

青が $\overrightarrow{AB}$、赤が逆ベクトル $-\overrightarrow{AB} = \overrightarrow{BA}$

④ベクトルの定数倍

実数 $k$ と $\vec{a}$ に対して $k\vec{a}$ を次のように定義する：

条件 $k\vec{a}$ の向き $k\vec{a}$ の大きさ
$k > 0$$\vec{a}$ と同じ向き$|k||\vec{a}|$
$k < 0$$\vec{a}$ と逆向き$|k||\vec{a}|$
$k = 0$零ベクトル$0$

加法と減法（足し算・引き算）

+ベクトルの加法（足し算）

三角形の法則

$\overrightarrow{AB} + \overrightarrow{BC} = \overrightarrow{AC}$

→ 終点と始点を合わせてつなぐと、残りの矢印が和になる

同じ始点から出発する場合（平行四辺形の法則）：$\vec{a} + \vec{b}$ は隣り合う辺とする平行四辺形の対角線になる。

法則

$\vec{a} + \vec{b} = \vec{b} + \vec{a}$ （交換法則）

$(\vec{a} + \vec{b}) + \vec{c} = \vec{a} + (\vec{b} + \vec{c})$ （結合法則）

−ベクトルの減法（引き算）

定義

$\vec{a} - \vec{b}$ は、$\vec{b}$ の逆ベクトルを足すことで計算する。

$$\vec{a} - \vec{b} = \vec{a} + (-\vec{b})$$

共通の始点 O を使うと：

$$\overrightarrow{AB} = \overrightarrow{OB} - \overrightarrow{OA}$$

AB = 終点 B − 始点 A と覚えよう。非常に重要な公式！

例ステップで確認：$\overrightarrow{AB} = \overrightarrow{OB} - \overrightarrow{OA}$

① $\overrightarrow{OA}$, $\overrightarrow{OB}$ を確認
② $-\overrightarrow{OA}$ を描く
③ $\overrightarrow{AB}$ を求める

No.2①

位置ベクトル

原点を基準にした点の表現と $\overrightarrow{AB}=\vec{b}-\vec{a}$

①位置ベクトルとは

定義

平面（または空間）上の固定した基準点 O（原点） を決め、各点 P の位置を 有向線分 $\overrightarrow{OP}$ で表したものを、
P の 位置ベクトル という。

$$\overrightarrow{OP} = \vec{p}$$

点 A の位置ベクトルを $\vec{a}$、点 B の位置ベクトルを $\vec{b}$ とする。

最重要公式
$$\overrightarrow{AB} = \vec{b} - \vec{a} \quad \text{（終点の位置ベクトル} - \text{始点の位置ベクトル）}$$

経路分解：$\overrightarrow{AB} = \overrightarrow{AO} + \overrightarrow{OB} = -\vec{a} + \vec{b} = \vec{b} - \vec{a}$

O を基準とした各点の位置ベクトルと、$\overrightarrow{AB}=\vec{b}-\vec{a}$ を図示

②位置ベクトルの利用

位置ベクトルを使うと、座標系を設定せずに図形の性質を証明・計算できる。

よく使う変換
  • $\overrightarrow{AB} = \vec{b} - \vec{a}$
  • $\overrightarrow{BA} = \vec{a} - \vec{b}$
  • $\overrightarrow{OA} = \vec{a}$（基本）
共線条件

3点 A, B, C が一直線上にある

$\Leftrightarrow\ \overrightarrow{AC} = k\,\overrightarrow{AB}$ を満たす実数 $k$ が存在する

$\Leftrightarrow\ \vec{c} = (1-k)\vec{a} + k\vec{b}$

例題

$\overrightarrow{OA}=\vec{a}$, $\overrightarrow{OB}=\vec{b}$ のとき、$\overrightarrow{AB}$, $\overrightarrow{BA}$ を求めよ。

解：$\overrightarrow{AB} = \vec{b}-\vec{a}$，$\overrightarrow{BA} = \vec{a}-\vec{b}$

分点と重心

内分点・外分点の位置ベクトルと三角形の重心

①内分点の位置ベクトル

内分点の公式

点 P が線分 AB を m : n に内分するとき、

$$\vec{p} = \frac{n\,\vec{a} + m\,\vec{b}}{m+n}$$

※「m に近い方（B側）が m 倍」と覚えると混乱しない

特別な場合：中点

線分 AB の 中点 M（1:1 内分）：

$$\vec{m} = \frac{\vec{a}+\vec{b}}{2}$$
1 1

②外分点の位置ベクトル

外分点の公式

点 Q が線分 AB を m : n に外分するとき（$m \neq n$）、

$$\vec{q} = \frac{-n\,\vec{a} + m\,\vec{b}}{m-n}$$

※ 内分の公式で $n$ を $-n$ に置き換えた形。分母が $m-n$ に変わることに注意。

例題

AB を 2:1 に外分する点 Q の位置ベクトルは？

解：$m=2, n=1$ を代入すると

$$\vec{q} = \frac{-1\cdot\vec{a}+2\cdot\vec{b}}{2-1} = 2\vec{b}-\vec{a}$$

③三角形の重心

重心の公式

△ABC の頂点の位置ベクトルを $\vec{a}, \vec{b}, \vec{c}$ とすると、重心 G の位置ベクトルは、

$$\vec{g} = \frac{\vec{a}+\vec{b}+\vec{c}}{3}$$

重心 = 3頂点の位置ベクトルの平均

△ABCの重心G（各中線の交点）の位置ベクトルは $\dfrac{\vec{a}+\vec{b}+\vec{c}}{3}$
内分点による導出

辺BCの中点Mの位置ベクトル：$\vec{m}=\dfrac{\vec{b}+\vec{c}}{2}$

GはAMを2:1に内分するから：

$$\vec{g}=\frac{1\cdot\vec{a}+2\cdot\frac{\vec{b}+\vec{c}}{2}}{3}=\frac{\vec{a}+\vec{b}+\vec{c}}{3}$$
△OABの重心（特別な場合）

Oが原点（$\vec{o}=\vec{0}$）のとき：

$$\vec{g}=\frac{\vec{0}+\vec{a}+\vec{b}}{3}=\frac{\vec{a}+\vec{b}}{3}$$
No.3①

位置ベクトルによる分解と点の位置

$\overrightarrow{AP}=\vec{p}-\vec{a}$ を使いこなそう

①ベクトルの分解（基点を変える）

基本変換

点 A を基点にとって位置ベクトル $\vec{b}=\overrightarrow{AB}$, $\vec{c}=\overrightarrow{AC}$ とおくと、

$$\overrightarrow{BC} = \overrightarrow{AC} - \overrightarrow{AB} = \vec{c} - \vec{b}$$

「終点の位置ベクトル − 始点の位置ベクトル」の原則と同じ。

一般形（原点Oを基準にするとき）

$\overrightarrow{OA}=\vec{a}$, $\overrightarrow{OP}=\vec{p}$ とすると、

$$\overrightarrow{AP} = \vec{p} - \vec{a} \quad \text{（終点 − 始点）}$$
A を基点とすると $\overrightarrow{BC}=\vec{c}-\vec{b}$（原点Oから見たベクトルの差）

②点の位置の決定

$m\overrightarrow{AP}+n\overrightarrow{BP}=\vec{0}$ のような式から点 P の位置ベクトルを求める。

手順
  1. $\overrightarrow{AP}=\vec{p}-\vec{a}$, $\overrightarrow{BP}=\vec{p}-\vec{b}$ のように「終点 − 始点」に直す
  2. $\vec{p}$ についての方程式を解く
  3. 求めた $\vec{p}$ が内分点・重心などと一致するか確認する
例題② $3\overrightarrow{AP}+\overrightarrow{BP}=\vec{0}$

$3(\vec{p}-\vec{a})+(\vec{p}-\vec{b})=\vec{0}$

$4\vec{p}=3\vec{a}+\vec{b}$

$$\vec{p}=\frac{3\vec{a}+\vec{b}}{4}$$

→ P は AB を 1:3 に内分する点（$n=3, m=1$）

例題③ $\overrightarrow{AP}+2\overrightarrow{BP}+3\overrightarrow{CP}=\overrightarrow{CA}$

$(\vec{p}-\vec{a})+2(\vec{p}-\vec{b})+3(\vec{p}-\vec{c})=\vec{a}-\vec{c}$

$6\vec{p}=2\vec{a}+2\vec{b}+2\vec{c}$

$$\vec{p}=\frac{\vec{a}+\vec{b}+\vec{c}}{3}$$

→ P は △ABC の 重心

重要パターン：$\overrightarrow{PA}+\overrightarrow{PB}+\overrightarrow{PC}=\vec{0}$

$(\vec{a}-\vec{p})+(\vec{b}-\vec{p})+(\vec{c}-\vec{p})=\vec{0}$ より $3\vec{p}=\vec{a}+\vec{b}+\vec{c}$

$$\vec{p}=\frac{\vec{a}+\vec{b}+\vec{c}}{3} \quad \Rightarrow \text{P は△ABCの重心}$$
No.3②

係数比較・平行条件

一次独立を使った方程式の解法と平行の判定

①一次独立と係数比較

一次独立（係数比較の定理）

$\vec{a}\neq\vec{0}$, $\vec{b}\neq\vec{0}$ で $\vec{a}$ と $\vec{b}$ が 平行でない（一次独立）とき、

$$m\vec{a}+n\vec{b}=m'\vec{a}+n'\vec{b} \iff m=m',\; n=n'$$

特に $m\vec{a}+n\vec{b}=\vec{0} \iff m=0$ かつ $n=0$

例題⑤ 係数比較で $x$, $y$ を求める

⑴ $3\vec{a}+x\vec{b}=y\vec{a}-4\vec{b}$

$\vec{a}$ の係数：$3=y$ → $y=3$

$\vec{b}$ の係数：$x=-4$

⑵ $(2x-5)\vec{a}+(4-3y)\vec{b}=\vec{a}-2\vec{b}$

$2x-5=1$ → $x=3$

$4-3y=-2$ → $y=2$

②ベクトルの平行条件

平行条件

$\vec{p}\neq\vec{0}$, $\vec{q}\neq\vec{0}$ のとき、

$$\vec{p}\parallel\vec{q} \iff \vec{q}=k\vec{p} \text{ となる実数 } k \text{ が存在する}$$
$\vec{q}=k\vec{p}$ のとき $\vec{p}\parallel\vec{q}$（$k>0$ なら同じ向き、$k<0$ なら逆向き）
例題⑥ $\vec{p}=2\vec{a}-3\vec{b}$, $\vec{q}=-6\vec{a}+x\vec{b}$ が平行になる $x$ は？

$\vec{q}=k\vec{p}$ とおくと $-6\vec{a}+x\vec{b}=k(2\vec{a}-3\vec{b})=2k\vec{a}-3k\vec{b}$

係数比較：$2k=-6 \Rightarrow k=-3$　また $x=-3k=9$

$$\boxed{x=9}$$
注意

$\vec{a}$, $\vec{b}$ が一次独立のとき $\vec{p}=s\vec{a}+t\vec{b}$, $\vec{q}=s'\vec{a}+t'\vec{b}$ であれば、

$\vec{p}\parallel\vec{q}$ $\iff$ $st'=s't$（係数の比が等しい）

No.4①

内積の定義と性質

ベクトルのかけ算「内積」— 結果はスカラー（数値）

①内積とは何か

ベクトルは「大きさ×向き」の量。2つのベクトルをかけ算するとスカラー（実数）が得られる。これを 内積 という。

（外積もあるが大学で学ぶ。高校では内積のみ。）

内積の定義

$\vec{a}$ と $\vec{b}$ のなす角を $\theta$（$0\le\theta\le\pi$）とするとき、

$$\vec{a}\cdot\vec{b} = |\vec{a}||\vec{b}|\cos\theta$$

この計算結果は 実数（スカラー） になる。

「なす角」の測り方

始点をそろえて 2つのベクトルをかいたとき、2つの方向がつくる角 $\theta$（$0\le\theta\le\pi$）が「なす角」。

2つのベクトルの始点をそろえて「なす角 $\theta$」を読み取る

計算例

例①　$|\vec{a}|=3,\ |\vec{b}|=2,\ \theta=\dfrac{\pi}{6}$
$$\vec{a}\cdot\vec{b}=3\cdot2\cdot\cos\frac{\pi}{6}=6\cdot\frac{\sqrt{3}}{2}=3\sqrt{3}$$
例②　$|\vec{a}|=4,\ |\vec{b}|=3,\ \theta=\dfrac{2\pi}{3}$
$$\vec{a}\cdot\vec{b}=4\cdot3\cdot\cos\frac{2\pi}{3}=12\cdot\left(-\frac{1}{2}\right)=-6$$
例③　$|\vec{a}|=5,\ |\vec{b}|=2,\ \theta=\dfrac{\pi}{2}$（垂直）
$$\vec{a}\cdot\vec{b}=5\cdot2\cdot\cos\frac{\pi}{2}=10\cdot0=0$$

→ 垂直なら内積は 必ず 0！

②内積の性質

内積の3つの性質

① 交換法則：$\vec{a}\cdot\vec{b}=\vec{b}\cdot\vec{a}$

② 分配法則：$\vec{a}\cdot(\vec{b}+\vec{c})=\vec{a}\cdot\vec{b}+\vec{a}\cdot\vec{c}$

③ 自己内積：$\vec{a}\cdot\vec{a}=|\vec{a}|^2$

③の証明：同じベクトルのなす角は $\theta=0$ なので、$\vec{a}\cdot\vec{a}=|\vec{a}||\vec{a}|\cos0=|\vec{a}|^2$。

例題：$|\vec{a}|=5,\ |\vec{b}|=2,\ \vec{a}\perp\vec{b}$ のとき各内積を求めよ

$\vec{a}\cdot\vec{a}$ $=|\vec{a}|^2=25$

$\vec{b}\cdot\vec{b}$ $=|\vec{b}|^2=4$

$\vec{a}\cdot\vec{b}$ $=0$（垂直）

$(2\vec{a}+\vec{b})\cdot\vec{b}$

$=2\vec{a}\cdot\vec{b}+\vec{b}\cdot\vec{b}$

$=2\cdot0+4=\mathbf{4}$

$(\vec{a}+\vec{b})\cdot(\vec{a}+\vec{b})$ $=|\vec{a}|^2+2\vec{a}\cdot\vec{b}+|\vec{b}|^2=25+0+4=\mathbf{29}$

③垂直条件

垂直条件（重要！）

$\vec{a}\neq\vec{0}$, $\vec{b}\neq\vec{0}$ のとき、

$$\vec{a}\perp\vec{b} \iff \vec{a}\cdot\vec{b}=0$$

なす角 $\theta=\dfrac{\pi}{2}$ のとき $\cos\frac{\pi}{2}=0$ となるため。

例題：$\vec{a}=\overrightarrow{OA}$, $\vec{b}=\overrightarrow{OB}$ として、$\overrightarrow{OA}\perp\overrightarrow{OB}$ を確認する

OA=3, OB=4, ∠AOB=90° のとき

$$\vec{a}\cdot\vec{b}=3\cdot4\cdot\cos\frac{\pi}{2}=0 \quad \checkmark$$
No.4②

内積の応用 — 角度・長さ・面積

内積を使って図形の計量問題を解く

①展開公式と余弦定理

内積は普通のかけ算と同じように展開できる：

展開公式（内積版）
$$|\vec{a}-\vec{b}|^2=|\vec{a}|^2-2\vec{a}\cdot\vec{b}+|\vec{b}|^2$$

証明：$(\vec{a}-\vec{b})\cdot(\vec{a}-\vec{b})=\vec{a}\cdot\vec{a}-2\vec{a}\cdot\vec{b}+\vec{b}\cdot\vec{b}$

余弦定理とのつながり

三角形OABで $\vec{a}=\overrightarrow{OA}$, $\vec{b}=\overrightarrow{OB}$, $\angle AOB=\theta$ とすると $|\vec{a}-\vec{b}|=AB$ なので、

$$AB^2=OA^2-2\vec{a}\cdot\vec{b}+OB^2=OA^2-2\cdot OA\cdot OB\cos\theta+OB^2$$

これはまさに 余弦定理！

$|\vec{b}-\vec{a}|^2$ の展開が余弦定理と一致する

②角度と面積の求め方

基本の3値を先にそろえる

2つの基準ベクトル $\vec{a}$, $\vec{b}$ をとったとき、まず

$$|\vec{a}|,\quad |\vec{b}|,\quad \vec{a}\cdot\vec{b}$$

の3値を求めれば、「長さ」「角度」「面積」はすべて計算できる。

なす角 $\theta$ を求める公式
$$\cos\theta=\frac{\vec{a}\cdot\vec{b}}{|\vec{a}||\vec{b}|}$$
面積を求める公式
$$S=\frac{1}{2}|\vec{a}||\vec{b}|\sin\theta=\frac{1}{2}\sqrt{|\vec{a}|^2|\vec{b}|^2-(\vec{a}\cdot\vec{b})^2}$$
例題① 辺の長さから内積・角度・面積を求める

三角形ABCで AB=4, BC=7, CA=5 とする。$\vec{a}=\overrightarrow{BA}$, $\vec{b}=\overrightarrow{BC}$ とおく。

（1）$\vec{a}\cdot\vec{b}$ を求める：

$|CA|=|\vec{b}-\vec{a}|=5$ より $|\vec{b}-\vec{a}|^2=25$

$|\vec{a}|^2-2\vec{a}\cdot\vec{b}+|\vec{b}|^2=25$　→　$16-2\vec{a}\cdot\vec{b}+49=25$

$$\vec{a}\cdot\vec{b}=20$$

（2）$\cos\angle ABC$ を求める：

$$\cos\angle ABC=\frac{\vec{a}\cdot\vec{b}}{|\vec{a}||\vec{b}|}=\frac{20}{4\times7}=\frac{5}{7}$$

（3）面積を求める：

$\sin^2\angle ABC=1-\dfrac{25}{49}=\dfrac{24}{49}$ → $\sin\angle ABC=\dfrac{2\sqrt{6}}{7}$

$$S=\frac{1}{2}\times4\times7\times\frac{2\sqrt{6}}{7}=4\sqrt{6}$$

③垂直条件の応用

例題② 直交する点の位置を求める

OA=4, OB=5, AB=6 の三角形OABで、辺AB上に点Pを $\overrightarrow{AC}\perp\overrightarrow{OP}$ となるようにとる。AP:PBを求めよ。

まず $|\vec{a}|=4$, $|\vec{b}|=5$ として $\vec{a}\cdot\vec{b}$ を求める。

$|{\vec{b}-\vec{a}}|^2=36$ より $16-2\vec{a}\cdot\vec{b}+25=36$ → $\vec{a}\cdot\vec{b}=\dfrac{5}{2}$

$\overrightarrow{OP}=(1-t)\vec{a}+t\vec{b}$ とおき、$\overrightarrow{AC}\perp\overrightarrow{OP}$ の条件 $\overrightarrow{AC}\cdot\overrightarrow{OP}=0$ を解くと

$$t=\frac{2}{5} \quad \Rightarrow \quad AP:PB=2:3$$
内積を使って「垂直になる点P」の位置を決定する
No.5①

ベクトルの成分表示

xy 座標を使ってベクトルを数値で表す

①成分表示とは

ベクトルの始点を原点においたときの 終点の座標 を「成分」という。

定義

x 軸方向の移動量を x 成分（$a_1$）、y 軸方向を y 成分（$a_2$）として、

$$\vec{a} = (a_1,\, a_2)$$

と表す。これを 成分表示 という。

2点間ベクトルの成分（重要！）
$$\overrightarrow{AB} = \text{（終点 B の座標）} - \text{（始点 A の座標）}$$

例）A$(2, 1)$, B$(5, 4)$ のとき $\overrightarrow{AB} = (5-2,\, 4-1) = (3, 3)$

$\vec{a}=(3,\,2)$：x方向に 3、y方向に 2 進むベクトル

②大きさの公式

三平方の定理から
$$|\vec{a}| = \sqrt{a_1^2 + a_2^2}$$

例）$\vec{a}=(3,\,4)$ のとき $|\vec{a}|=\sqrt{9+16}=\sqrt{25}=5$

2点間の距離

A$(x_1,\,y_1)$, B$(x_2,\,y_2)$ のとき

$$AB = |\overrightarrow{AB}| = \sqrt{(x_2-x_1)^2+(y_2-y_1)^2}$$

③成分による演算

加法・減法・実数倍

$\vec{a}=(a_1,a_2)$, $\vec{b}=(b_1,b_2)$, $k$ を実数とするとき

$$\vec{a}+\vec{b}=(a_1+b_1,\,a_2+b_2)$$
$$\vec{a}-\vec{b}=(a_1-b_1,\,a_2-b_2)$$
$$k\vec{a}=(ka_1,\,ka_2)$$

→ 対応する成分どうしで計算するだけ！

例題：A$(-3,\,1)$, B$(-1,\,-2)$, C$(0,\,2)$ のとき

$\overrightarrow{AB}$ $=(-1-(-3),\,-2-1)=(2,-3)$

$\overrightarrow{AC}$ $=(0-(-3),\,2-1)=(3,1)$

$|\overrightarrow{AB}|$ $=\sqrt{4+9}=\sqrt{13}$

$|\overrightarrow{AC}|$ $=\sqrt{9+1}=\sqrt{10}$

平行四辺形の条件

四角形 ABDC が平行四辺形 $\iff$ $\overrightarrow{AB}=\overrightarrow{CD}$

「同じ成分のベクトルが等しい」→ 各成分ごとに方程式を立てて解く

No.5②

成分による内積・角度・面積

成分が分かれば内積・角度・面積が計算できる

①成分による内積の公式

基本ベクトル $\vec{e}_1=(1,0)$, $\vec{e}_2=(0,1)$ の内積は、

$$\vec{e}_1\cdot\vec{e}_1=1,\quad \vec{e}_2\cdot\vec{e}_2=1,\quad \vec{e}_1\cdot\vec{e}_2=0$$

これを利用して $\vec{a}=(a_1,a_2)$, $\vec{b}=(b_1,b_2)$ の内積を展開すると、

内積の成分公式（重要！）
$$\vec{a}\cdot\vec{b} = a_1 b_1 + a_2 b_2$$

→ x 成分どうし・y 成分どうしをかけて 足すだけ！

例① $\vec{a}=(1,2)$, $\vec{b}=(3,4)$
$$\vec{a}\cdot\vec{b}=1\times3+2\times4=11$$
例② $\vec{a}=(2,-1)$, $\vec{b}=(1,3)$
$$\vec{a}\cdot\vec{b}=2\times1+(-1)\times3=-1$$
成分から直接内積を計算する（なす角θがわからなくても OK）

②角度・面積の計算

成分表示での cos θ
$$\cos\theta = \frac{\vec{a}\cdot\vec{b}}{|\vec{a}||\vec{b}|} = \frac{a_1b_1+a_2b_2}{\sqrt{a_1^2+a_2^2}\,\sqrt{b_1^2+b_2^2}}$$
面積（成分版）
$$S = \frac{1}{2}\sqrt{|\vec{a}|^2|\vec{b}|^2 - (\vec{a}\cdot\vec{b})^2}$$
例題：A$(2,0)$, B$(4,-1)$, C$(8,2)$ — 角度と面積を求める

$\overrightarrow{AB}=(2,-1)$, $\overrightarrow{AC}=(6,2)$ として

内積： $\overrightarrow{AB}\cdot\overrightarrow{AC}=2\times6+(-1)\times2=10$

大きさ： $|\overrightarrow{AB}|=\sqrt{5}$, $|\overrightarrow{AC}|=\sqrt{40}=2\sqrt{10}$

$$\cos\angle BAC=\frac{10}{\sqrt{5}\cdot2\sqrt{10}}=\frac{10}{2\sqrt{50}}=\frac{10}{10\sqrt{2}}=\frac{1}{\sqrt{2}} \;\Rightarrow\; \angle BAC=45°$$
$$S=\frac{1}{2}\sqrt{5\times40-100}=\frac{1}{2}\sqrt{100}=5$$

③直線のベクトル方程式と垂線の足

直線のベクトル方程式

点 A を通り、方向ベクトル $\vec{d}$ に平行な直線上の点 P は、

$$\overrightarrow{OP} = \overrightarrow{OA} + t\vec{d} \quad (t \in \mathbb{R})$$
例題：A$(2,1)$, $\vec{d}=(2,3)$ — B$(-2,0)$ から直線に下ろした垂線の足 H

直線上の点 H を $\overrightarrow{OP}=\overrightarrow{OA}+t\vec{d}=(2t+2,\,3t+1)$ とおく。

$\overrightarrow{BH}\perp\vec{d}$ より $\overrightarrow{BH}\cdot\vec{d}=0$：

$$\overrightarrow{BH}=(2t+4,\,3t+1)$$
$$(2t+4)\times2+(3t+1)\times3=4t+8+9t+3=13t+11=0 \;\Rightarrow\; t=-\frac{11}{13}$$
$$H\!\left(\frac{4}{13},\,-\frac{20}{13}\right)$$
$\overrightarrow{BH}\perp\vec{d}$ の条件：$\overrightarrow{BH}\cdot\vec{d}=0$ で H の位置を決定
